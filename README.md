# BMW-Sales-2010-2024-Data-Analysis-Report
BMW Sales Analysis (2010–2024) – A practice data analysis project exploring BMW’s global sales trends. Covers regional performance, model profitability, pricing impact, fuel-type shifts, and customer preferences using Python, Power BI, and Excel for actionable business insights.
Here is the original link to the Dataset used (Kaggle): https://www.kaggle.com/datasets/ahmadrazakashif/bmw-worldwide-sales-records-20102024
